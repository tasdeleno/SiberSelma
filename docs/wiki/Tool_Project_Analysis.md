# Proje Analiz Aracı (Tool Project Analysis)

**Özet:** Verilen yerel projelerin kaynak kodlarını okuyup, LLM için statik analiz (SAST) bağlamı oluşturan MCP aracıdır.
**Kütüphaneler:** Python built-in.
**Bağlantılar:** [[Index]], [[Server_Core]]

## Detaylar

Şu anki kod tabanında (`analyze_project_vulnerabilities` fonksiyonu) henüz detaylı bir kaynak kod okuma işlemi yapılmamaktadır (Placeholder/Taslak durumundadır).
Amacı: Hedef dizindeki `.js`, `.py`, `.ts` gibi dosyaları okuyup içindeki potansiyel tehlikeli kod örüntülerini LLM'in dikkatine sunmaktır.
